# Shopsmart-Digital-Grocery-Store

Full stack Grocery Store Web Application:-



## Overview
ShopSmart is an online grocery web app for a seamless shopping experience. Customers can browse products, manage their cart, and place orders. Admins can manage products, users, and orders.

---

## Features

### Frontend
- Intuitive and responsive UI
- Product catalog with search, filters, and sorting
- Shopping cart and multi-step checkout
- User authentication and profile management
- Payment integration (Stripe/PayPal)

### Backend
- Node.js + Express.js server
- RESTful APIs for users, products, orders
- MongoDB database integration
- Secure authentication and data management

### User Roles
- **Admin:** Manage products, users, orders, and generate reports  
- **User:** Browse products, manage cart, place orders, view order history

---

## Project Flow
1. Frontend (Angular)  
2. Backend (Node.js, Express.js, MongoDB)  
3. Integration and testing
